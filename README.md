1. Copy the 'robot' folder inside the ics-os/ics-os/contrib directory.
    $cd ics-os/ics-os/contrib
    $make
    $make install

2. Perform the steps to install ics-os.
   $cd ../..
   $sudo make install

3. Run robot.exe inside ics-os
    $cd apps
    $robot.exe
